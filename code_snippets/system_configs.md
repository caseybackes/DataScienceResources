# System Configurations and Hacks

#### Alias for 'cd ..' to move muliple levels

Add the following alias to the .bash_profile through vi(m) or nano.

`alias ..="cd .."`

`alias ...="cd ../.."`

`alias ....="cd ../../.."`

Save the file, exit the text editor, and re-source the .bash_profile

`$ source .bash_profile`
